## Why sh-exec?

Use `Template literals` write shell script made happy ❤️.

## Installation

```sh
$ yarn add sh-exec
```

## Examples

```js
import sh from 'sh-exec'
import { version } from '../package.json'

sh`
echo "sh-exec is awesome."
`

sh`
git init
git add .
git commit -m '${version}'
`
```

## Promise Based

```js
sh`
  curl https://a-url.com
`
  .then((stdout) => {
    console.log('done')
  })
  .catch((err) => {
    console.error(err)
  })

// or
;(async () => {
  const stdout = await sh`
    curl https://a-url.com
  `
})()
```
